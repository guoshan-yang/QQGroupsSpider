## QQ群信息的抓取，基于python3.6
批量抓取 QQ 群信息，包括群名称、群号、群人数、群主、地域、分类、标签、群简介等内容，返回 XLS / CSV / JSON 结果文件。

## 运行
```
python3 qq_groups_spider.py
```
